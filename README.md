#  Sobre o Projeto

Este projeto apresenta um estudo de implementação de serviços da **Amazon Web Services (AWS)** na empresa fictícia **Abstergo Industries**, com foco em:

- 💰 Redução de custos operacionais
- ⚡ Aumento da disponibilidade dos sistemas
- 🔒 Segurança da infraestrutura
- 📈 Escalabilidade sob demanda
- 🌐 Conectividade híbrida corporativa

O relatório demonstra como diferentes soluções AWS podem transformar ambientes locais tradicionais em arquiteturas modernas baseadas em computação em nuvem.

# 🏗️ Arquitetura e Serviços Utilizados

## ☁️ Amazon EC2

### Objetivo
Migrar aplicações corporativas para instâncias virtuais escaláveis.

### Benefícios
- Escalabilidade sob demanda
- Redução de custos com hardware
- Melhor aproveitamento de recursos
- Alta disponibilidade

### Caso de Uso
A empresa (ficticia ) utilizava servidores físicos com alto custo de manutenção.  
Com o Amazon EC2, as aplicações passaram a operar em ambientes virtualizados e escaláveis dentro da AWS.

---

## 🔐 Amazon VPC

### Objetivo
Criar uma rede privada segura dentro da AWS.

### Benefícios
- Isolamento de sistemas internos
- Controle de acesso
- Sub-redes privadas
- Comunicação segura com VPC Endpoints

### Caso de Uso
Deverá ser criada uma infraestrutura privada para proteger os recursos corporativos e evitar exposição direta à internet pública.

---

## 🌐 AWS Direct Connect

### Objetivo
Estabelecer conectividade híbrida dedicada entre o datacenter corporativo e a AWS.

### Benefícios
- Redução de latência
- Maior estabilidade de conexão
- Menor custo de transferência
- Comunicação via fibra óptica Ethernet

### Caso de Uso
A empresa passa a utilizar conexão dedicada entre o ambiente on-premises e a AWS para aplicações críticas.
